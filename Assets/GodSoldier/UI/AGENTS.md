# UI Agent Guide

- The menu and lobby should feel premium, not placeholder-flat.
- Main menu structure:
- left rail for `Play`, `Settings`, `Exit`
- right side for live 3D scene framing and atmospheric copy
- animated mission timeline after pressing `Play`
- Lobby structure:
- mission briefing
- public/private match mode choice
- role lock for `God` and `Soldier`
- session status and ready/start flow
- Avoid player-facing use of `slice`.
